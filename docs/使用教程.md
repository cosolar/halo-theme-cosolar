# halo-theme-cosolar 主题使用与配置手册

> 一款为开发者打造的现代 Halo 博客主题：青绿美学、明暗双模、精选轮播、动态背景、全局搜索、完美移动端适配。
>
> 适用版本：`>= 2.20.0`（主题版本 1.0.7）

本手册以主题后台 **设置（cosolar-setting）** 中的全部配置项为基准，逐项说明每个字段的含义、默认值、取值范围与配置建议；并补充菜单图标、文章注解、依赖插件等「设置界面之外」的实用功能。读完即可独立完成从安装到精调的全过程。

## 1. 安装与启用

| 项目 | 要求 |
| ---- | ---- |
| Halo | `>= 2.20.0` |

**方式一：上传 ZIP（推荐）**

1. 下载最新 `cosolar-<version>.zip`
2. 后台 **主题管理 → 安装主题** → 上传 ZIP
3. 安装完成后点击 **启用**

**方式二：源码构建**

```bash
pnpm install
pnpm build            # 产物：templates/ + cosolar-<version>.zip
```

将生成的 ZIP 上传到 Halo，或将整个目录放入 Halo 工作目录的 `themes/cosolar/` 下。

## 2. 配置入口与基本概念

启用主题后，进入 **Halo 后台 → 主题管理 → halo-theme-cosolar → 设置**，即可看到按分组排列的所有配置项。

> **两个重要约定**
> - **留空即默认**：绝大多数文本 / 图片 / 颜色字段留空时，主题会使用内置默认效果（如默认 Logo、默认青绿渐变背景、默认字体等）。不确定时可以先留空观察效果。
> - **明暗双模独立**：背景、博主卡片、页脚、登录页等视觉项普遍提供「浅色 / 深色」两套配置，切换主题时自动套用对应的一套，互不影响。

下文表格中字段名（如 `primary_color`）为内部键名，便于高级用户对照 `../settings.yaml`；日常配置只需关注「配置项」列。

## 3. 基础设置（basic）

配置站点身份信息，显示在导航栏与全局。

| 配置项 | 字段名 | 类型 | 默认值 | 说明与建议 |
| --- | --- | --- | --- | --- |
| Logo 图片 | `logo_image` | 附件 | 空 | 导航栏左侧 Logo；留空显示 `</>` 代码图标。建议透明背景 PNG，尺寸 36–80px |
| Logo 图片宽度 | `logo_image_width` | 数字 | 36 | 显示宽度（px），范围 20–80，步长 2 |
| Logo 图片高度 | `logo_image_height` | 数字 | 36 | 显示高度（px），范围 20–80，步长 2 |
| Logo 文字 | `logo_text` | 文本 | `码迹志` | 导航栏站点名称，2–6 个字最佳 |
| 站点标语 | `tagline` | 文本 | `记录技术成长，分享解决方案` | Logo 下方的副标题 |
| 页脚版权文字 | `footer_text` | 多行文本 | 空 | 页脚底部版权信息，**支持 HTML**。如 `© 2026 我的博客 · All rights reserved` |
| ICP 备案号 | `icp` | 文本 | 空 | 国内站点备案号；填写后自动链接工信部网站，留空不显示 |

## 5. 主题样式（style）

控制主题色、明暗方案、布局尺寸与字体图标。

| 配置项 | 字段名 | 类型 | 默认值 | 说明与建议 |
| --- | --- | --- | --- | --- |
| 主题色 | `primary_color` | 颜色 | `#10B981` | 全站按钮 / 链接 / 标签 / 进度条 / 选中态联动变色 |
| 默认配色方案 | `color_scheme` | 单选 | `auto`（跟随系统） | 可选：跟随系统 / 浅色 / 暗色；用户仍可用导航栏按钮手动切换 |
| 内容最大宽度 | `content_width` | 数字 | 1600 | 范围 960–1920，文字博客建议 1100–1200，图片类可加大 |
| 侧边栏宽度 | `sidebar_width` | 数字 | 320 | 范围 260–400，内容多可调到 360 |
| 内容间距 | `content_gap` | 数字 | 24 | 内容区与侧边栏间距，范围 12–48 |
| 文章目录宽度 | `toc_width` | 数字 | 240 | 右侧 TOC 宽度，范围 180–320 |
| 文章缺省封面图 | `default_post_cover` | 附件 | 空 | 文章无封面时默认图，建议 16:9 横图；留空显示代码图标占位符 |
| 正文字体族 | `font_family` | 文本 | 空 | 自定义 `font-family`，如 `"Noto Sans SC", sans-serif`；留空用默认霞鹜文楷 |
| 字体 CSS 地址 | `font_css_url` | URL | 空 | 第三方字体 CSS，如 Google Fonts；留空用默认霞鹜文楷 CDN |
| 等宽字体族 | `mono_family` | 文本 | 空 | 代码字体，如 `"Source Code Pro", monospace`；留空用 JetBrains Mono |
| 等宽字体 CSS 地址 | `mono_css_url` | URL | 空 | 第三方等宽字体 CSS，留空不额外加载 |
| 图标字体 CSS 地址 | `icon_css_url` | URL | 空 | 第三方图标库 CSS（Remix Icon / Font Awesome 等）；留空用内置 iconfont |
| 图标类名前缀 | `icon_prefix` | 文本 | `iconfont` | 内置 `iconfont`；Font Awesome 用 `fa-solid`/`fas`，Remix 用 `ri`，Lucide 用 `lucide` |

> 代码高亮由 Halo `plugin-shiki` 插件负责（后台配置亮 / 暗主题），主题本身不提供该选项。

## 7. 首页轮播（featured）

首页顶部精选文章轮播卡片。

| 配置项 | 字段名 | 类型 | 默认值 | 说明与建议 |
| --- | --- | --- | --- | --- |
| 启用精选轮播 | `show_featured` | 开关 | 开 | 关闭则首页不显示轮播 |
| 轮播默认背景图 | `featured_cover` | 附件 | 空 | 文章无封面时的背景；留空用主题默认背景 |
| 优先使用文章封面 | `featured_cover_fallback` | 开关 | 关 | 开启后优先用文章自带封面，否则用默认背景图 |
| 精选文章列表 | `featured_posts` | 文章选择 | 空 | 多选、可拖拽排序，最上一张显示在第一位 |
| 自动轮播 | `autoplay` | 开关 | 开 | 开启后自动切换 |
| 轮播间隔（毫秒） | `interval` | 数字 | 5000 | 最小 1000，步长 500 |
| 轮播最大文章数 | `featured_max_count` | 数字 | 5 | 回退模式下最多显示数量，范围 1–10 |
| 无配置时的回退策略 | `fallback` | 单选 | `latest`（最新） | 可选：显示最新文章 / 显示置顶文章 / 不显示 |
| 首页加载方式 | `home_load_mode` | 单选 | `pagination`（分页） | 分页：点击翻页；滚动加载：滚动到底自动加载（保留「查看更多文章」兜底） |

> 想让某篇文章进轮播：在「精选文章列表」直接选它；或把文章**置顶**并选回退策略为「显示置顶文章」。

| 显示热门标签 | `show_tag_cloud` | 开关 | 开 | 词云，字体大小随文章数变化 |
| 热门标签数量 | `tag_cloud_count` | 数字 | 20 | 范围 5–50 |
| 显示公众号二维码 | `show_wechat_qr` | 开关 | 关 | 开启后展示二维码 |
| 公众号二维码图片 | `wechat_qr_image` | 附件 | 空 | 建议 1:1 正方形 |
| 公众号名称 | `wechat_qr_title` | 文本 | 空 | 留空显示「扫码关注」 |

| 显示近期更新 | `show_recent` | 开关 | 开 | — |
| 近期更新数量 | `recent_count` | 数字 | 4 | 范围 1–30 |
| 显示友情链接模块 | `show_sidebar_links` | 开关 | 开 | 依赖 `plugin-links` |
| 友情链接显示数量 | `sidebar_links_count` | 数字 | 5 | 范围 1–30 |
| 显示订阅资讯模块 | `show_sidebar_feeds` | 开关 | 开 | 依赖 `plugin-links` 公开 RSS |
| 订阅资讯显示数量 | `sidebar_feeds_count` | 数字 | 5 | 范围 1–30 |

**默认模块顺序（从上到下）**

`博主展示 → 公告 → 热门标签 → 公众号 → 分类导航 → 近期更新 → 友情链接 → 订阅资讯`

> 各模块自身的「显示 XX 模块」开关用于临时显隐，不影响已设定的顺序；数组里删除某行则彻底隐藏该模块。

| 页面标题 | `feeds_title` | 文本 | `RSS 资讯` | — |
| 页面副标题 | `feeds_subtitle` | 文本 | `聚合优质信息源，掌握技术前沿动态` | 标题下方描述 |
| 页面封面图 | `feeds_cover` | 附件 | 空 | Hero 背景，留空用默认渐变 |
| 每页资讯数量 | `feeds_page_size` | 数字 | 10 | 范围 5–50，步长 5，建议 10–30 |

> 创建独立页面时，页面类型选择 **RSS 资讯（feeds）**。页面为三栏布局：左栏频道导航、中栏资讯流（加载更多分页）、右栏概览 / 活跃频道 / 抓取状态。

## 14. 页面背景 · 浅色模式（background_light）

浅色模式下的整站背景，三种类型互斥。

| 配置项 | 字段名 | 类型 | 默认值 | 说明与建议 |
| --- | --- | --- | --- | --- |
| 背景类型 | `bg_type` | 下拉 | `none`（默认纯色） | 可选：无 / 自定义背景图 / 动态渲染背景 |
| 背景图 | `bg_image` | 附件 | 空 | 选「自定义背景图」时生效，建议浅色调低饱和 |
| 背景图透明度 | `bg_image_opacity` | 滑块 | 0.2 | 范围 0.05–1 |
| 动态背景类型 | `bg_dynamic_type` | 下拉 | `particles`（粒子连线） | 选「动态渲染背景」时生效，见下方 5 种 |
| 动态背景主色 | `bg_dynamic_color1` | 颜色 | `#10B981` | — |
| 动态背景辅色 | `bg_dynamic_color2` | 颜色 | `#3B82F6` | 用于渐变 / 混合 |
| 动态背景透明度 | `bg_dynamic_opacity` | 滑块 | 0.15 | 范围 0.05–0.5，建议 0.05–0.3 |

**5 种动态背景效果**

| 类型值 | 效果 | 适合场景 |
| --- | --- | --- |
| `particles` 粒子连线 | 随机粒子，相近连线 | 科技 / 开发者主页 |
| `gradient` 渐变流光 | 径向渐变光斑漂移 | 个人 / 文艺 |
| `bubbles` 气泡漂浮 | 半透明气泡上升 | 轻松活泼 |
| `stars` 星空闪烁 | 星点明暗交替 | 深色 / 夜间 |
| `mesh` 网格波纹 | 网格点阵 + 波纹 | 极简几何 |

## 15. 页面背景 · 深色模式（background_dark）

与浅色模式完全独立，配置方式一致，默认动态背景为「星空闪烁」。

| 配置项 | 字段名 | 类型 | 默认值 | 说明与建议 |
| --- | --- | --- | --- | --- |
| 背景类型 | `bg_type` | 下拉 | `none` | 同浅色模式 |
| 背景图 | `bg_image` | 附件 | 空 | 建议深色调 |
| 背景图透明度 | `bg_image_opacity` | 滑块 | 0.2 | 范围 0.05–1 |
| 动态背景类型 | `bg_dynamic_type` | 下拉 | `stars`（星空闪烁） | 推荐与深色背景搭配 |
| 动态背景主色 | `bg_dynamic_color1` | 颜色 | `#10B981` | — |
| 动态背景辅色 | `bg_dynamic_color2` | 颜色 | `#3B82F6` | — |
| 动态背景透明度 | `bg_dynamic_opacity` | 滑块 | 0.15 | 范围 0.05–0.5 |

> 两种模式背景互不影响，切换主题时平滑淡入淡出。动态背景用 Canvas 2D 渲染，粒子数随屏幕自动调整（最多约 80 粒子 / 150 星星），对现代设备几乎无性能影响。

## 19. 依赖插件与搜索

**友情链接 / RSS 资讯页依赖 `plugin-links`**

| 插件 | 用途 | 必需度 |
| --- | --- | --- |
| `plugin-links` | 管理友链、检测可达性与回链、聚合友站 RSS / Atom | 友链页、侧边栏友链 / 订阅模块、RSS 资讯页**必需** |

未安装时表现：友链页列表为空、侧边栏「友情链接」「订阅资讯」模块自动隐藏、RSS 资讯页显示「未启用」提示。安装后还需在插件「RSS 订阅」中开启「公开 RSS 订阅动态」，RSS 资讯页与侧边栏订阅模块才有内容。

**搜索功能依赖官方搜索插件**

1. 安装 **PluginSearchWidget** 插件
2. 导航栏自动显示搜索框，支持 `Ctrl+K` 快捷键
3. 未安装时搜索框降级为归档页链接

**代码高亮依赖 `plugin-shiki`**

在后台插件设置中配置亮 / 暗主题，主题本身不提供该选项。

## 20. 常见问题

**如何完全离线部署？**
下载字体文件上传到 Halo 附件管理，在「主题样式」将「字体 CSS 地址」替换为本地路径；图标字体已内置，无需额外处理。

**动态背景影响性能吗？**
Canvas 2D 渲染，粒子数随屏幕自动调整，几乎无影响；如顾虑可降低透明度或切静态背景图。

**为什么友链 / RSS 页没有内容？**
确认已安装并启用 `plugin-links`，并在插件「RSS 订阅」开启公开动态；未装插件时相关模块自动隐藏。

**侧边栏模块顺序怎么调？**
在「侧边栏设置 → 侧边栏模块顺序」拖拽每行调整；删除某行隐藏该模块，可重加。各模块单独开关用于临时显隐，不影响顺序。

**博主背景图和背景色区别？**
背景图适合深色图（文字自动变白 + 半透明遮罩）；背景色更简洁；同时设置时背景图优先。

**响应式断点**

| 断点 | 布局 | 行为 |
| --- | --- | --- |
| `> 1024px` | 三栏 | 内容 + 侧边栏 + TOC |
| `769–1024px` | 双栏 | 内容 + 侧边栏（隐藏 TOC） |
| `≤ 768px` | 单栏 | 侧边栏隐藏、汉堡菜单、操作栏贴底 |
| `≤ 480px` | 紧凑单栏 | 按钮间距进一步收窄 |
